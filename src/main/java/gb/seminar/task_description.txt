Задание 2.
Обработайте возможные исключительные ситуации. “Битые” значения в исходном массиве считайте нулями. Можно внести в код
правки, которые считаете необходимыми.
public class Main {
    public static int sum2d(String[][] arr) {
        int sum = 0;
        for (int i = 0; i < arr.length; i++) {
            for (int j = 0; j < 5; j++) {
            int val = Integer.parseInt(arr[i][j]);
            sum += val;
            }
        }
        return sum;
    }
}
